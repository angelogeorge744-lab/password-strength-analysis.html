# 🔐 Password Strength Analyzer

An interactive web application that analyzes password strength using:

- Combinatorics (N = C^L)
- Entropy (bits)
- Brute-force simulation
- Probability theory

## 🚀 Features
- Real-time password analysis
- Entropy calculation
- Pattern detection
- Attack simulation
- Interactive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
